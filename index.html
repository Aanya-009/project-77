My first Console Project


Visual Studio is an offline editor. It is very easy and fun to use. It is also very easy to set it up. You just need to download the file given by the website and then follow the steps to set it up.


Console screen is a system window which displays data which is given by the programmer. To open the console screen you need to right click on a webpage and the select inspect. It will open a window. Choose console screen. To print anything on the console screen we use the command console.log();
